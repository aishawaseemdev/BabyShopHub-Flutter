BabyShopHub-Flutter
A complete Flutter e-commerce application designed for baby products, featuring real-time data integration with Firebase.

Overview
BabyShopHub is an e-commerce platform that allows users to browse and view baby products. The application is built using Flutter and leverages Firebase Firestore for dynamic, real-time product fetching and management.

Key Features
Dynamic Product Listing: Fetches featured products directly from Firebase Firestore.

Modern UI/UX: Built with a responsive and sleek design.

Firebase Integration: Ensures seamless backend connectivity for data management.

Tech Stack
Frontend: Flutter (Dart)

Backend: Firebase (Firestore)

Project Status
Completed as a semester project for Aptech Computer Education NN2.
